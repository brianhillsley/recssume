  • Metal 3D Printer Java API

  • Python Web Crawler

    I designed and coded a Python web crawler that looks for suspicious strings within webpages. The crawler emulates the activity of a web browser and has timeouts to avoid getting stuck. While crawling, the application produces graphs and a database of hashcoded results.

  • Aquam: Undersea Adventure Video Game

    An undersea adventure video game built in Java without the use of a game engine. The game consists of 6 different levels with over 30 different enemies with custom graphics and sound.
